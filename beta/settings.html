ServiceDesk / Support Center Timer - Settings


General information
After you click "save" button, please reload the main webpage!
This timer is still WIP, if you encounter any bug, please create an issue :)!
Connect URI
"Connect URI" - needs to be exact address of the webpage. Example: "https://supportcenter.example.org/" with the "/" at the end!
If you want the feature to verify if timelog was added - add this webpage to the headers(trusted origin) or use CORS.
API-KEY
"API-KEY" - API KEY generated from the ServiceDesk Plus or the SupportCenter Plus or both
UserID
"USERID" - you need to obtain it from the system and WebConsole.
1.) Login to the system
2.) Open WebConsole(with F12)
3.) Type "userID"
4.) Get the variable value and paste it here
